# Quantum-Safe Federated IDS for 6G Edge Networks

## 🚀 Project Overview
This project implements a **Quantum-Safe Federated Intrusion Detection System (IDS)** designed for 6G Edge environments. It addresses the critical need for privacy-preserving security that can withstand both classical and post-quantum cryptographic threats.

## 🔑 Key Features
* **Federated Learning:** Privacy-preserving model training using the **Flower (Flwr)** framework.
* **Post-Quantum Security:** Integration of **Kyber-512** (NIST finalist) for quantum-resistant model updates.
* **Edge Optimization:** Lightweight 1D-CNN architecture optimized for low-latency edge devices.
* **IDS Capabilities:** Detects DoS, DDoS, Injection, and Scanning attacks with **91.8% accuracy**.

## 🛠️ Tech Stack
* **Language:** Python 3.10
* **ML Engine:** TensorFlow / Keras
* **Federation:** Flower (v1.7.0)
* **Cryptography:** liboqs (Open Quantum Safe)
